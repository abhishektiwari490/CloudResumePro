CloudResume Pro – Cloud Native Portfolio Platform

CloudResume Pro is a premium cloud-native engineering portfolio designed to showcase technical projects while simultaneously demonstrating practical AWS and DevOps skills. Unlike traditional portfolios, this project itself follows a production-inspired architecture by integrating static hosting, serverless computing, Infrastructure as Code and deployment automation.

The frontend is developed using HTML, CSS and JavaScript with a focus on responsive design and modern user experience principles. The application is intended to be hosted using AWS S3 Static Website Hosting and distributed globally through CloudFront, ensuring high availability and low-latency content delivery.

Future backend services will be implemented using AWS Lambda, API Gateway and DynamoDB to provide visitor analytics, dynamic content management and serverless API integration. Terraform will be used to provision infrastructure while GitHub Actions will automate deployment, creating a complete end-to-end Cloud & DevOps project.

☁️ Cloud File Storage System

Developed a secure cloud-native file storage solution using AWS services to provide scalable and reliable file upload, storage and retrieval capabilities. The system is designed around a serverless architecture, reducing infrastructure management overhead while maintaining high security standards.

AWS S3 is used for durable object storage, while AWS Lambda generates pre-signed URLs that allow secure file uploads without exposing cloud credentials. API Gateway acts as the API layer and IAM policies ensure controlled access to cloud resources.

📱 NHRMERS – Smart Healthcare Management System

NHRMERS (National Health Record Management and Emergency Response System) is a cloud-integrated Android application developed to simplify secure patient data management and improve access to healthcare information. The application combines modern Android development practices with cloud-backed services to create a scalable healthcare platform.

The application leverages Firebase Authentication for secure login and account management, while Firebase Realtime Database enables synchronized storage and retrieval of patient records. The system architecture is designed to support future integration with telemedicine services, emergency response modules and cloud-based healthcare analytics.
